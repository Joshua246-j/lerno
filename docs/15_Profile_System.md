# 15 - Profile System

## 👤 User Identity
The profile is the user's digital identity within Lerno. Children can customize it extensively using earned coins.

## 📊 Statistics Tracking
The profile prominently displays:
- Total Games Played.
- Win Rate (%).
- Current League Standing.
- Total XP & Coins.

## 🎒 Inventory
Users have a digital inventory containing:
- **Avatars**: Selectable profile pictures (Unlockable or Purchasable).
- **Borders**: Decorative frames for avatars.
- **Titles**: Suffixes or prefixes attached to the username (e.g., "Math Whiz").

## ⚙️ Settings
The settings module, accessible from the profile, allows customization of:
- BGM & SFX volume levels.
- Theme preferences (Light/Dark mode).
- Notification toggles.
- Account management (Logout, Delete Account) - *Note: Destructive actions require parental pin validation.*
