# Engagement & Retention Strategy

A high-quality learning platform is useless if users churn after day three. Lerno utilizes industry-standard gaming retention strategies to build long-term habits.

## 🔄 The Core Loop

1. **Learn**: User completes a 3-minute lesson (Drag-and-drop, matching, or Flame mini-game).
2. **Earn**: User receives XP (progresses on the Learning Path) and Coins (virtual currency).
3. **Spend**: User spends Coins in the Store for cosmetic Avatar upgrades or chat stickers.
4. **Compete**: User equips the new Avatar and joins a 1v1 Quiz Battle to show it off and earn League Trophies.

## 📅 Daily Retention Mechanics

- **Daily Missions**: "Complete 3 Math Lessons", "Win 1 Battle". Yields a large bonus chest of Coins.
- **Login Streaks**: Consecutive daily logins multiply the XP yield, creating a sunk-cost fallacy that encourages returning.
- **Push Notifications**: Personalized, playful alerts generated by the backend (e.g., "Your friend Alex just passed you in the Silver League!").

## 🏆 Long-Term Retention

- **Leagues**: A 7-day cyclical tournament. The top 20% in a league promote, the bottom 20% demote. This triggers massive engagement peaks on Sundays before the reset.
- **Content Updates**: New Learning Paths and mini-games unlock seasonally to prevent content exhaustion.
